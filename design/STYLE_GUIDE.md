# Velvet Hour Audit - Style Guide

## Brand Identity
**Name:** Velvet Hour Audit
**Vibe:** Premium, Professional, Actionable, High-end.
**Tagline:** Agency-level insights for small businesses, in an hour.

## Color Palette
- **Velvet Primary:** `#3A0CA3` (Deep Royal Purple)
- **Velvet Secondary:** `#7209B7` (Vibrant Purple)
- **Gold Accent:** `#FFD700` (Luxury Highlight)
- **Champagne:** `#F3E5AB` (Soft highlight)
- **Deep Slate:** `#212529` (Text and Dark Mode backgrounds)
- **Ghost White:** `#F8F9FA` (Main backgrounds)
- **Success Green:** `#4CC9F0` (Actually a bright blue, let's go with `#2ECC71` for success)
- **Alert Red:** `#E63946`

## Typography
- **Headings (Display):** `Playfair Display`, Serif.
- **Sub-headings:** `Montserrat`, Sans-Serif (Bold).
- **Body Text:** `Inter`, Sans-Serif.
- **Monospace (for code/technical bits):** `JetBrains Mono`.

## UI Components
- **Buttons:** Rounded corners (8px), Primary is Velvet Primary with White text. Hover state: Velvet Secondary.
- **Cards:** White background, subtle shadow (`box-shadow: 0 4px 6px -1px rgb(0 0 0 / 0.1)`), 12px border radius.
- **Inputs:** Simple borders, focus state uses Gold Accent glow.

## Imagery
- High-quality, professional photography.
- Abstract velvet textures for backgrounds.
- Clean, minimalist icons (Phosphor Icons or Lucide).
